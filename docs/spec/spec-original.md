# 개요
기본적으로 다음 기능을 생각하고 있다.

- 실시간 통신 매치메이킹 가능 (일단 전체 30분으로 턴마다 이를 사용하게끔 함. 10초 미만이면 10초가 되도록 채움. 시간이 모두 종료되면 시간사)
- 솔로플레이 기능(봇 존재)
- 회원가입 필수 (OAuth 2.0, 자체 로그인 기능)
- 커뮤니티 존재
- 상대와 채팅 가능
- 카드깡 기능
- 마이 페이지 (덱 변경, 일부 설정 변경)

게임은 다음 방식으로 진행한다.

1. 카드에 따른 행동을 한다.
    1. 카드를 사용한다.
    2. 아무것도 하지 않는다.
2. 자신 턴에 말을 움직인다.
3. 카드에 따른 행동을 한다.
    1. 카드를 사용한다.
    2. 일반 카드를 뽑는다. (덱은 최대 5개)
    3. 아무것도 하지 않는다.
4. 턴 종료

카드는 레전드 카드, 일반 카드 두 종류가 있고, 처음에 레전드 카드 1개, 일반 카드 1개를 가지고 시작한다. 일반 카드는 상대도 볼 수 있다.

React + Spring + WebSocket 기반이다.
